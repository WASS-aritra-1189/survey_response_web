├─ Title: 22px
└─ Height: ~420px

## Animation States

```
Page Load:
Card slides up from bottom with fade-in (0.6s)

Button Hover:
Lifts up 2px with shadow increase (0.3s)

Button Click:
Returns to normal position (0.1s)

Loading:
Spinner rotates continuously (1s)

Error/Success:
Message slides down with fade-in (0.3s)

Background:
Decorations float up and down (6s infinite)
```

## Accessibility Features

```
✅ Keyboard Navigation
   Tab → Next field
   Shift+Tab → Previous field
   Enter → Submit form

✅ Focus Indicators
   2px outline on focused elements
   High contrast (4.5:1 minimum)

✅ Screen Reader
   Semantic HTML
   ARIA labels
   Descriptive text

✅ Color Contrast
   Text on White: 7.1:1 (WCAG AAA)
   Text on Blue: 8.5:1 (WCAG AAA)
   Error Red: 5.2:1 (WCAG AA)
```
